# rizzma image tests

This directory holds the scaffolding for rizzma's matplotlib golden-image
tests (plan PR-13).

## Layout

- `tests/baselines/` — reference PNGs generated by upstream matplotlib. These
  are the "golden" images that rizzma renders are checked against.
- rizzma renders are produced at test time (typically under `target/`) and are
  not committed.

## Comparing images

The `xtask` automation crate provides a root-mean-square image comparison:

```text
cargo xtask image-diff <baseline.png> <render.png> [--tolerance <f64>] [--out <diff.png>]
```

- The comparison loads both images as RGBA and errors if their dimensions
  differ.
- It reports the root-mean-square per-channel difference (0-255 scale) and the
  maximum absolute single-channel difference.
- It passes when the RMS value is within `--tolerance` (default `2.0`) and
  exits non-zero on failure, so it can gate CI.
- When `--out` is given, an absolute per-pixel difference image is written for
  visual inspection of regressions.

Example:

```text
cargo xtask image-diff tests/baselines/line_plot.png target/render/line_plot.png \
    --tolerance 1.5 --out target/diff/line_plot.png
```
